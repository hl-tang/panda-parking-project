# panda-parking-project
![image](https://github.com/taizogt/panda-parking-project/assets/131645131/40e65085-c949-4530-9e81-12c2c5392cc5)

## 概要
2024.3.16-3.17 の 2 日間で開催された Track Jobs 主催のハッカソンイベント．

本チームは，Vue.js + Node.js を用いた Web アプリケーションを開発．

### 開発テーマ

駐輪場を管理する団体 (地方自治体，学校，賃貸管理者) による駐輪場管理のDX化.
- 管理者
  - 契約情報 (契約期間，駐輪番号) の管理をシールから本アプリへ
  - 保有する駐輪場の情報一覧 (場所，契約料金，利用率 etc) の確認 
  - 保有する駐輪場の，駐輪場毎の利用率・利益の可視化
- 自転車保有者
  - 本アプリを用いた，駐輪場契約情報の管理 (追加・削除)
  - 1 ID に契約情報を紐づけることによる管理の簡素化
<img src="https://github.com/taizogt/panda-parking-project/assets/131645131/6918fd12-bf74-46ce-aca4-b9503db9b129" width="400">

### 使用技術
- フロントエンド: Vue.js (Vuetify, Vite, Echart, ...)
- バックエンド: Node.js
- APIプラットフォーム: Postman
- DB: MongoDB
